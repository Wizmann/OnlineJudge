# 极值问题1

# Description
有`N`个自然数，它们的和是`M`，问这`N`个自然数的平方和最大是多少？最小是多少？

# Format

## Input

第一行输入T，第2~T+1行每行各有两个整数N和M。

* 1 <= 100 <= T
* 1 <= N <= 1000
* 1 <= M <= 10^9

## Output

输出T行，每行各有两个正整数，代表每组数据的平方和最大值与最小值。

输出对`1000000007`取模。

# Sample 1

## Input
```
1
11 2021
```

## Output
```
4084441 371315
```
